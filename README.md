## About

Welcome to my personal portfolio website!

This project showcases my journey as a **Frontend Developer** and **Java Developer**, highlighting my technical skills, featured projects, and passion for building responsive, accessible, and modern web applications.

The website was designed from scratch using semantic HTML, modern CSS, and JavaScript while following responsive design principles and accessibility best practices.
